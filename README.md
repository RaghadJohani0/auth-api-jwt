# Secure JWT Authentication API

## Features
- JWT authentication (Access & Refresh Tokens)
- HttpOnly Cookies
- Role-based Authorization
- Secure password hashing (bcrypt)
- Protected Routes
- Logout & Token Rotation

## Tech Stack
- Node.js
- Express
- JWT
- PostgreSQL / MongoDB
